<Info>
  **Prerequisites**:

  - Node.js version 19 or higher
  - PNPM installed 
</Info>

Follow these steps to install and run the OpenCourseWare Platform on your operating system.

<Steps>
  <Step title="Clone the repo">
    ```bash
    git clone https://not-ani/ocw-rewrite.git
    ```
  </Step>
  <Step title="Preview locally">
    

    ```bash
    cd ocw-rewrite && pnpm i 
    ```
  </Step>
  <Step title="Set Convex Up" stepNumber={3}>
    ```bash
    pnpm dev:setup
    ```
  </Step>
  <Step title="Fill out the env file">
    ```
    NEXT_PUBLIC_CONVEX_URL=
    NEXT_PUBLIC_CLERK_FRONTEND_API_URL=
    NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
    CLERK_SECRET_KEY=
    NEXT_PUBLIC_POSTHOG_KEY=
    NEXT_PUBLIC_POSTHOG_HOST=""
    ```
  </Step>
  <Step title="Run the server">
    ```bash
    pnpm dev
    ```
  </Step>
</Steps>

By default, the app runs on port 3001.
